# Security Log Analyzer

Python-based log analysis tool that detects failed login attempts and identifies suspicious IP addresses.

Features:
Reads log files
Detects failed logins
Counts IP occurrences
Flags suspicious activity
Sorts by highest failed attempts

Technologies:
Python
Dictionaries
File handling

Example Output:
--- Failed Login Summary ---

192.168.1.5, -> 2 failed login attempts
10.0.0.8, -> 1 failed login attempts
